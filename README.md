## 前言

欢迎来到我们的Java计算机毕业设计项目：springboot汽车资讯网站。本项目是一个集成了多种前沿技术的实战项目，适用于毕业设计或学习交流。以下是对本项目的详细介绍。

## 内容介绍

本项目基于Spring Boot框架，使用Java开发，前端采用JS、Vue和css3技术，数据库采用MySQL 5.7/8.0。本项目旨在为用户提供一个全面的汽车资讯平台，包括汽车品牌、车型、配置、价格等方面的信息。通过本项目，用户可以轻松获取到所需的汽车资讯，提高了用户体验。

## 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、css3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
